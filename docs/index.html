🛠️ Technical Implementation

🔍 Vector Search Engine

Built with HNSWlib for efficient approximate nearest neighbor search with sub-millisecond query times.

🧠 Embedding Generation

Uses PyTorch & SentenceTransformers to generate 768-dimensional semantic embeddings from research papers.

⚡ FastAPI Backend

RESTful API built with FastAPI providing endpoints for embedding generation and similarity search.

☁️ Cloud Deployment

Deployed using Docker, Kubernetes, AWS S3 & Lambda for scalable and efficient operations.
